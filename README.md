일주일에 두 개씩 각자 알고리즘을 풀어서 공유하자

### 정책
  - GitHub을 이용해서 알고리즘 정리 내용을 넣자.
  - GitHub의 MarkDown 사용
  - LeetCode를 중심으로 시작! (책, 유투브의 내용들도 가능)
  - 매 미팅 시 중복되지 않게 알고리즘을 각자 선택.
  - 난이도는 무관
  - 매주 월요일 진행
