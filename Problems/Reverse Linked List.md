#### Merge Two Sorted Lists
#### Writer: Haram Ryu
#### Refer: LeetCode

## 문제 설명
두 개의 linked list가 입력된다. 각각의 linked list는 숫자 오름차순으로 정렬이 된 상태. 입력된 두 linked list를 병합된 하나의 linked list로 나타내되 역시 숫자 오름차순으로 정렬하여 병합된 하나의 linked list로 나타내어야한다.
***예시***
```
Input: 1->2->4, 1->3->4
Output: 1->1->2->3->4->4
```
