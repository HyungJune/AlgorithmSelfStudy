#### Merge Two Binary Trees
#### Writer: Haram Ryu
#### Refer: LeetCode

## 문제 설명
주어진 두개의 바이너리 트리를 루트 노드를 겹쳐서 놓은 다음 같은 노드에 위치한 값들끼리는 더해서 새로운 바이너리 트리로 만들어서 출력  
***예시***
```
Input: 
	Tree 1                     Tree 2                  
          1                         2                             
         / \                       / \                            
        3   2                     1   3                        
       /                           \   \                      
      5                             4   7                  
Output: 
Merged tree:
	     3
	    / \
	   4   5
	  / \   \ 
	 5   4   7
 ```
